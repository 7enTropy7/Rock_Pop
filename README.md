## Cloning
```bash
$ git clone https://github.com/7enTropy7/Rock_Pop.git
```

## Dependencies
```bash
$ pip3 install -r requirements.txt
```

## Demonstration

```bash
$ python3 Pytorch_DDPG/testing.py
```

For Single Player:
```bash
$ python3 main.py
```
